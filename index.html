FAQs

Have Questions?

Is this like Tornado Cash?

No. ShieldTX is a compliant trading solution for Hyperliquid traders looking to protect against copy-bots, competitors and adversaries.

What pairs are supported?

Every pair available on Hyperliquid is supported. If there's a specific pair you'd like added, get in touch.

What is the maximum leverage available?

The maximum leverage on ShieldTX is the same as Hyperliquid.

Does shielding affect my fills?

No. Execution is native HL: same order book, same liquidity. There is no slippage premium for shielding.

Who controls the vault / my funds?

You do. Funds are locked in a smart contract on Arbitrum. Core operations are permissionless and verified by zero-knowledge proofs, no admin key required. See the full breakdown in the trust model.

Is there an API?

API access is on the roadmap post-MVP. If you're running algo strategies, reach out directly. Early access conversations are open.
